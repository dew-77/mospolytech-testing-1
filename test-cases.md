# Тест-кейсы для калькулятора

## TC-001: Сложение двух чисел
Шаги: Открыть калькулятор. Нажать "5", затем "+", затем "3", затем "="
Ожидаемый позитивный результат: На экране отображается "8"
Ожидаемый негативный результат: Отображается другое число или ошибка, калькулятор зависает

## TC-002: Вычитание
Шаги: Открыть калькулятор. Нажать "9", затем "−", затем "4", затем "="
Ожидаемый позитивный результат: На экране отображается "5"
Ожидаемый негативный результат: Неверный результат или ошибка вычисления

## TC-003: Умножение
Шаги: Открыть калькулятор. Нажать "6", затем "×", затем "7", затем "="
Ожидаемый позитивный результат: На экране отображается "42"
Ожидаемый негативный результат: Отображается неверный результат или NaN

## TC-004: Деление с десятичным результатом
Шаги: Открыть калькулятор. Нажать "7", затем "÷", затем "2", затем "="
Ожидаемый позитивный результат: На экране отображается "3.5"
Ожидаемый негативный результат: Ошибка округления (например, 3.4999999), потеря точности или некорректный формат

## TC-005: Работа с отрицательными числами
Шаги: Открыть калькулятор. Нажать "5", затем "±", затем "+", затем "8", затем "="
Ожидаемый позитивный результат: На экране отображается "3"
Ожидаемый негативный результат: Неверный знак результата, некорректная обработка отрицательного числа

## TC-006: Деление на ноль
Шаги: Открыть калькулятор. Нажать "10", затем "÷", затем "0", затем "="
Ожидаемый позитивный результат: На экране отображается "Ошибка", калькулятор продолжает работать после очистки
Ожидаемый негативный результат: Отображается Infinity, NaN, приложение зависает или крашится

## TC-007: Множественные десятичные точки
Шаги: Открыть калькулятор. Нажать "3", затем ".", затем "5", затем "." (второй раз)
Ожидаемый позитивный результат: Вторая точка не добавляется, на экране остается "3.5"
Ожидаемый негативный результат: Добавляется вторая точка (например, "3.5."), ошибка ввода или краш

## TC-008: Ведущие нули
Шаги: Открыть калькулятор. Нажать "0", затем "0", затем "0", затем "5"
Ожидаемый позитивный результат: На экране отображается "5" (не "0005")
Ожидаемый негативный результат: Отображается "0005", ошибка форматирования или некорректная обработка

## TC-009: Последовательные операторы
Шаги: Открыть калькулятор. Нажать "5", затем "+", затем "−", затем "×"
Ожидаемый позитивный результат: Последний оператор заменяет предыдущий, на экране "5" и оператор "×"
Ожидаемый негативный результат: Операторы накапливаются, ошибка вычисления или зависание

## TC-010: Цепочка вычислений
Шаги: Открыть калькулятор. Нажать "10", затем "+", затем "5", затем "=", затем "×", затем "2", затем "="
Ожидаемый позитивный результат: Сначала на экране "15", затем "30"
Ожидаемый негативный результат: Потеря промежуточного результата, неверный финальный результат или ошибка в цепочке вычислений
